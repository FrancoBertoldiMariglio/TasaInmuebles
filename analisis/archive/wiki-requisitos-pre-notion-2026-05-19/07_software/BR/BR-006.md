# BR-006 — Escala ordinal estado_conservacion + coeficientes Fitt-Servini

## Enunciado

> El campo `estado_conservacion` de toda [[T-026]] Tasación es un **entero en escala 1-5** que mapea biunívocamente a un coeficiente decimal usado en la fórmula [[T-011]] Fitt-Servini (RF-016) para calcular el [[T-030]] Valor técnico.

## Tabla biunívoca

| `estado_conservacion` | Etiqueta UI | Coeficiente Fitt-Servini |
|---|---|---|
| **5** | A estrenar | **1.00** |
| **4** | Muy bueno | **0.85** |
| **3** | Bueno | **0.70** |
| **2** | Regular | **0.55** |
| **1** | A refaccionar | **0.40** |

## Función

```
coef_conservacion = lookup(tabla_BR006, estado_conservacion)

donde lookup(_, 5) = 1.00
      lookup(_, 4) = 0.85
      lookup(_, 3) = 0.70
      lookup(_, 2) = 0.55
      lookup(_, 1) = 0.40
```

## Reglas derivadas

- Validación en RF-005: `estado_conservacion ∈ {1, 2, 3, 4, 5}`. Cualquier otro valor rechaza la persistencia.
- En UI: control con 5 opciones (slider 1-5 con etiquetas, o radio buttons).
- El coeficiente entra en BR-019 (fórmula Fitt-Servini lite).

## Parámetros configurables

| Parámetro | Valor actual | Configurable | Quién cambia |
|-----------|--------------|--------------|--------------|
| Coeficientes `[1.00, 0.85, 0.70, 0.55, 0.40]` | fijos en MVP | Sí (Fase 2 vía Back Office) | Admin S-023 + arquitecto del Colegio |
| Etiquetas UI de los 5 niveles | fijas en MVP | Sí (i18n Fase 2-3) | Admin S-023 |

## Fuente de autoridad

- **Fitt y Servini** (manual estándar argentino de tasación técnica). Coeficientes son los de uso convencional, a ajustar con un arquitecto del Colegio (vinculado a DS-08).
- Cierre A-016 [2026-05-14, Franco]: confirmación de escala 1-5.

## Impacto si cambia

- Cambio de coeficientes: afecta directamente todos los cálculos Fitt-Servini desde la fecha del cambio. Versionar fórmula (BR-019 v2). Tasaciones previas siguen con sus valores ya calculados (snapshot persiste en `valor_tecnico_inputs`).
- Cambio de escala (ej: pasar a 1-7): cambio mayor de UI + BR-006 v2 + RF-005 v2 + migración de datos existentes.

## Por qué Cálculo (con Hecho secundario)

- **Cálculo dominante**: la regla se usa operativamente como **lookup determinístico** para entrar a la fórmula Fitt-Servini.
- **Hecho secundario**: "existen 5 niveles de conservación" es un hecho del dominio, pero su importancia operativa es el cálculo.
- Tipo dominante registrado: **Cálculo**.
