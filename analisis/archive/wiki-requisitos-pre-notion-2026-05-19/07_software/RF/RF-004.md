# RF-004 — Capturar datos del solicitante

## Descripción
El sistema captura los datos de [[T-032]] Solicitante de la [[T-026]].

## Campos
- `solicitante.nombre` — string, **requerido**.
- `solicitante.apellido` — string, **requerido**.
- `solicitante.telefono` — string formato E.164 con prefijo de país (default ARG, +54), **requerido**.
- `solicitante.email` — string formato email, **opcional** (pero requerido si se va a usar RF-014 Enviar PDF por mail).

## Validación
- Email valida formato RFC si presente.
- Teléfono valida formato E.164.
- Nombre y apellido: no vacíos.

## UI
- Acordeón "Solicitante" en formulario "Nueva tasación" (ver imagen `12.45.44(3).jpeg`).
- Selector de país con bandera + prefijo (default Argentina).

## Pregunta pendiente
- Q11: nomenclatura UI (Solicitante vs Referente).
