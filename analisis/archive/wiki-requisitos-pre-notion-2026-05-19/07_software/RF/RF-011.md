# RF-011 — Listar tasaciones del tasador ("Mis tasaciones", pantalla principal app mobile)

## Descripción
El sistema muestra la lista de [[T-026]] del [[T-028]] autenticado, ordenada por fecha descendente, con filtros básicos. Esta es la **[[T-033]] "Mis tasaciones"** — pantalla principal de la app mobile, equivalente al `home` de la app. **No confundir con el dashboard del Owner (T-034 web / T-035 mobile embebida)**, que es Fase 2.

## Flujo
1. Tasador entra a la app mobile → se abre directo en [[T-033]] "Mis tasaciones".
2. Sistema consulta BD: `SELECT * FROM tasaciones WHERE tasador_id = current_user ORDER BY fecha_alta DESC`.
3. Sistema renderiza cards/filas con: ID, domicilio, tipo, motivo, estado (badge color), fecha alta, valor (si tasada).
4. Tasador puede filtrar por: estado, tipo_inmueble, motivo, rango de fechas.
5. Tasador puede tocar una fila → ficha de tasación (CU-UI similar a imagen `12.45.44(1)`).
6. Tasador puede tocar "+ Nueva tasación" → dispara CU-UI-001.

## Validación
- Solo se listan tasaciones del usuario autenticado (no se exponen tasaciones de otros tasadores).
- Filtros componen con AND.
- Paginación: 20 por página MVP.

## UI MVP-6sem
- **Solo app mobile** (no hay versión web del tasador en MVP).
- Lista vertical con cards. Touch-first.

## Fuera del MVP-6sem
- Dashboard del Owner versión web (CU-UI-010, T-034).
- Dashboard del Owner versión mobile primitiva embebida (T-035).
- KPIs avanzados (mes vs mes anterior, zonas, etc.).
- Multi-tenant selector.
