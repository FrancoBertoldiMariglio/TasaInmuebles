# BR-019 — Cálculo: Fórmula Fitt-Servini lite

## Enunciado

> El [[T-030]] Valor técnico de una [[T-026]] Tasación se calcula aplicando la **fórmula Fitt-Servini lite v1**:

```
superficie_terreno = max(superficie_total_m2 - superficie_cubierta_m2, 0)

valor_construccion = superficie_cubierta_m2
                   × valor_m2_categoria[tipo_inmueble][categoria]
                   × coef_conservacion[estado_conservacion]   # BR-006

valor_terreno = superficie_terreno
              × valor_m2_terreno[zona]
              × coef_frente_fondo                              # = 1.0 en MVP

valor_tecnico_USD = valor_construccion + valor_terreno
valor_tecnico_ARS = valor_tecnico_USD × tipo_cambio_referencia
```

## Parámetros configurables

| Parámetro | Origen | Configurable MVP | Configurable Fase 2 |
|-----------|--------|------------------|----------------------|
| `valor_m2_categoria[tipo][categoria]` | Tabla DS-08 (arquitecto del Colegio + Cocucci) | Hardcoded en `config/fitt_servini_table.json` | Vía Back Office (CU-UI-017) |
| `valor_m2_terreno[zona]` | Tabla DS-08 | Hardcoded JSON | Vía Back Office |
| `coef_conservacion[1..5]` | BR-006 | Hardcoded `[0.40, 0.55, 0.70, 0.85, 1.00]` | Vía Back Office |
| `coef_frente_fondo` | Inferido geometría | Fijo = 1.0 en MVP | Cálculo dinámico Fase 2 desde lat/lng + manzana catastral |
| `tipo_cambio_referencia` | Manual (admin) | Carga manual mensual | API BCRA o dolarapi.com Fase 2 |

## Versionado

- `version_formula = "fitt-servini-lite-v1"` se persiste en cada cálculo (campo `valor_tecnico_version_formula` de la tasación).
- Cambios futuros incrementan versión: `v2` (Fase 2 con coef frente/fondo dinámico), `v3` (Fase 3 con depreciación no-lineal), etc.
- Las tasaciones viejas conservan la versión con la que fueron calculadas (auditoría).

## Reglas derivadas

- Snapshot de inputs en cada cálculo: `valor_tecnico_inputs` (JSON con todos los valores usados). Permite recalcular o auditar.
- Si zona no está en `valor_m2_terreno[zona]`: aplica fallback `valor_m2_zona_default` + flagea en PDF "valor estimado con datos genéricos por falta de zona específica".
- Si el resultado da 0 o negativo: error de cálculo → log + alerta a admin.

## Fuente de autoridad

- **Fitt y Servini**: manual estándar argentino de tasación técnica usado por arquitectos colegiados.
- Versión "lite" simplificada para MVP: omite coeficiente frente/fondo dinámico y depreciación no-lineal (Fase 2-3).
- Parametrización de tabla: DS-08 (pendiente bloqueante MVP).

## Impacto si cambia

- **Cambio de tabla `valor_m2_*`**: las tasaciones nuevas usan la tabla nueva. Las viejas conservan sus valores (snapshot).
- **Cambio de coeficientes** (BR-006): mismo principio, snapshot preserva.
- **Cambio de fórmula** (v2): se incrementa `version_formula`. Tasaciones viejas siguen siendo válidas con su versión.

- No hay condición (Habilitador), restricción (Restricción), ni inferencia probabilística (Inferencia).
- Es entrada → función pura → salida. Cálculo puro.
