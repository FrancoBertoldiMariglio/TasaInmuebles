# BR-002 — Habilitador de matrícula vigente para Tasador profesional

## Enunciado

> Solo un usuario con `role = tasador` cuya **matrícula profesional esté vigente y verificada** puede:
> - Iniciar sesión en la app mobile (RF-010).
> - Crear, editar o cerrar una [[T-026]] Tasación profesional.
> - Participar en el [[T-006]] Comité de tasación.
> - Firmar profesionalmente una tasación certificada (Fase 2).

## Condición habilitante

- `usuario.matricula_vigente = true` (verificada manualmente por admin en MVP-6sem; automatizable contra API del Colegio en Fase 2-3).
- `usuario.role = tasador` (vs `cliente_b2c`).
- `usuario.curso_uso_plataforma_completado = true` (mencionado por Cocucci en transcript:L171 — en MVP es presentación informal por Cocucci, en Fase 2 puede ser módulo formal).

## Verificación

- **MVP-6sem**: admin (Franco/Cocucci) verifica matrícula a ojo durante CU-UI-006 (pre-carga manual). Persiste `matricula_numero` + `matricula_vigente=true` en BD.
- **Fase 2-3**: integración con API del Colegio de Arquitectos para validación automática (depende de que el Colegio exponga API — DS-05).

## Fuente de autoridad

- Reglamento profesional argentino (Colegios de Arquitectos provinciales).
- Acuerdo Cocucci ↔ Colegio (BR-NEG-003).
- Transcripción reunión-01 L171: "cualquiera que nosotros vamos a verificar que esté matriculado".

## Impacto si cambia

- Si se levanta el requisito de matrícula (improbable): caería el modelo Uber de tasación basado en profesionales matriculados.
- Si se agrega verificación automática (Fase 2): se reduce trabajo manual del admin y se previene fraude.

## Por qué Habilitador (no Restricción)

- **Habilitador**: define la condición para que algo **se permita** (positiva: matrícula vigente + curso → puede tasar).
- **Restricción** sería: "no puede iniciar sesión sin matrícula" (negativa).
- Acá la formulación natural es positiva (qué se necesita para habilitar la acción), por eso clasifica como Habilitador.

## Diferencia con Cliente B2C

- Esta BR **NO aplica** a [[T-037]] Cliente B2C. Los Clientes B2C no necesitan matrícula (su flujo es la [[T-036]] Tasación referencial sin firma profesional). Para ellos aplica BR-022.
