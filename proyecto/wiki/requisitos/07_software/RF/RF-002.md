# RF-002 — Capturar motivo de tasación

## Descripción
El sistema permite al [[T-028]] seleccionar el `motivo` de un dropdown con el catálogo cerrado de 10 opciones.

## Flujo
1. Sistema muestra dropdown con: Venta · Alquiler · Asesoramiento de valores · Tasación para empresas · Expropiaciones · Divisiones/fraccionamientos · Sucesión · Tasación judicial · Tasación extrajudicial · Hipoteca.
2. Tasador selecciona uno.
3. Sistema persiste `motivo` en el draft.

## Validación
- Selección obligatoria para guardar.
- Catálogo cerrado (BR-005 / CBR-017).
- **Nota**: validación legal por tipo (oficio judicial para "judicial", etc.) es **Fase 2** — fuera del MVP.

## Entrada / Salida
- IN: `motivo` ∈ catálogo de 10
- OUT: campo persistido en draft.
