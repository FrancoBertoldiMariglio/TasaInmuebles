# RF-006 — Subir fotos del inmueble

## Descripción
El sistema permite al [[T-028]] subir entre 1 y 15 fotos del inmueble desde la cámara del dispositivo o galería.

## Flujo
1. Tasador toca acordeón "Fotos".
2. Sistema muestra grid de slots (hasta 15) con botón "+".
3. Tasador toca "+" → sistema invoca cámara o file picker.
4. Tasador captura o selecciona N fotos.
5. Sistema sube cada foto al storage (S3, Cloudinary o local — pendiente decisión técnica).
6. Sistema muestra thumbnails en orden de subida.
7. Tasador puede reordenar arrastrando (post-MVP).
8. Tasador puede eliminar una foto antes de guardar.

## Validación
- Mínimo 1 foto requerida para guardar (RF-008).
- Máximo 15 fotos por tasación (BR-007).
- Tamaño máx por foto: pendiente AC (sugerencia 5 MB).
- Formatos permitidos: JPEG, PNG, HEIC.
- Si el tasador supera 15, el botón "+" se deshabilita y muestra "Máximo 15 fotos".

## UI MVP
- Carrusel simple en consulta (no AirBnB-style en MVP).
- En captura: grid 3×5 con slots vacíos visibles.

## Errores
- E-001: foto excede tamaño → rechazar + sugerir comprimir.
- E-002: pérdida de conexión durante upload → mantener foto en draft local, reintentar.
