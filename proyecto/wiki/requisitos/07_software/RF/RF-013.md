# RF-013 — Descargar PDF (opción A de "Compartir")

## Descripción
El sistema entrega el PDF generado por RF-012 al dispositivo del [[T-028]] para descarga local.

## Flujo
1. Tasador en la ficha de la tasación (estado ≥ Tasada) toca "Descargar PDF".
2. Sistema verifica que el PDF existe en storage (si no, lo genera con RF-012).
3. Sistema responde con header `Content-Disposition: attachment; filename="Tasacion-{id}-{domicilio}.pdf"`.
4. Browser/Capacitor descarga.
5. Sistema persiste registro: `download_event(tasacion_id, tasador_id, timestamp)`.
6. Sistema transiciona la tasación a [[T-007]] Compartida (si era la primera descarga o envío).

## Seguridad
- Solo el `tasador_id` autor o miembros del comité pueden descargar.
- JWT requerido en el endpoint.

## Auditoría
- `download_event` queda persistido para auditoría futura.
