# RF-001 — Seleccionar tipo de inmueble

## Descripción
El sistema permite al [[T-028]] seleccionar el `tipo_inmueble` de un catálogo cerrado de 6 opciones al iniciar una nueva [[T-026]].

## Flujo
1. Sistema muestra chips horizontales (UI mobile) con: Casa · Departamento · Terreno · Galpón · Local · Oficina.
2. Tasador toca una opción.
3. Sistema persiste `tipo_inmueble` en el draft.
4. Si tipo es Terreno/Galpón/Local/Oficina, sistema oculta los campos `dormitorios` y `banos` de RF-005 (no aplican).

## Validación
- Selección obligatoria para guardar (RF-008).
- Catálogo cerrado: no se admiten otros valores (BR-004).

## Entrada / Salida
- IN: `tipo_inmueble` ∈ {casa, departamento, terreno, galpón, local, oficina}
- OUT: campo persistido en draft de tasación.
