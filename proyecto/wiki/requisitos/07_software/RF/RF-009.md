# RF-009 — Transicionar a "Por tasar" / enviar a comité

## Descripción
Al guardar exitosamente (RF-008), el sistema transiciona la [[T-026]] automáticamente al estado [[T-022]] **Por tasar** para que sea visible al [[T-006]].

## Flujo
1. RF-008 completa con éxito → estado = `Inspección hecha`.
2. Sistema dispara automáticamente transición a `Por tasar` (BR-015 / CBR-015).
3. Sistema notifica a los miembros del comité (en MVP: vía dashboard, no push ni mail).
4. La tasación aparece en la lista "Por tasar" del dashboard del comité.

## Postcondición
- Estado de la tasación = `Por tasar`.
- Tasación visible en dashboard de comité.
- `fecha_envio_comite` = timestamp.

## Fuera del MVP-6sem
- Notificación push al comité.
- Mail automático al comité.
- Tiempo medio en cola "Por tasar" como métrica del dashboard.
